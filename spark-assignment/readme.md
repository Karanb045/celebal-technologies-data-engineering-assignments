# Apache Spark Basics Assignment

## Overview

This repository contains the implementation of basic **Apache Spark** operations using **PySpark DataFrames**. The project demonstrates how to load, clean, transform, filter, aggregate, and analyze data using Spark.

The assignment also includes theoretical and practical questions for **Week 5**, covering Spark fundamentals and DataFrame operations.

---

## Objectives

* Understand the differences between Hadoop MapReduce and Apache Spark.
* Learn how Spark performs in-memory processing.
* Load CSV datasets into Spark DataFrames.
* Perform data cleaning and preprocessing.
* Apply filtering and transformations.
* Perform aggregations and grouping operations.
* Build a complete Spark data processing pipeline.

## Assignment Tasks

The notebook demonstrates:

* Viewing schema and sample records

* Renaming columns
