
# Shopping Dataset Analysis

## Objective

Perform exploratory data analysis (EDA), data cleaning, and derive meaningful insights from the combined dataset.

## Dataset

**`combined_dataset.csv`** file from https://www.kaggle.com/datasets/anvitkumar/shopping-dataset
* Records: 1000
* Features: 26

## Tasks Performed

* Data Loading and Inspection
* Missing Value Analysis
* Duplicate Record Check
* Data Cleaning
* Feature Engineering
  * Price Difference
  * Popularity Score
* Univariate Analysis
* Bivariate Analysis
* Category-wise Analysis
* Data Visualization

## Key Findings

* Average product rating is 3.62.
* Fashion categories such as Tops, Dresses, and Shirts dominate the dataset.
* Discounts average 53.5%.
* Product popularity is influenced more by ratings and review counts than by discounts.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
